Random Rotation vs Randomized Hadamard Transform

2D 벡터에 두 변환을 적용 — 같은 입력, 다른 방법, 비슷한 결과

Random Rotation

x̃ = Rx
입력 x 출력 Rx 단위원

Randomized Hadamard (RHT)

x̃ = (1/√d) H D x
입력 x 출력 HDx/√d Dx (중간)

R (랜덤 직교행렬)

O(d2) 저장

D (대각행렬)

O(d) 저장 - 부호만

H (Hadamard)

저장 불필요 - 구조 고정

(1/√d) HD

= RHT 등가 행렬
